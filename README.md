# 🏛 AI Economy Simulator

**ИИ-симулятор экономики** — 1000 автономных ИИ-агентов моделируют экономику в реальном времени. Никакого сценария — агенты сами принимают решения, торгуют, создают институты и обучаются.

## Концепция

- **1000 агентов** стартуют на равных условиях с одинаковым капиталом
- **Валюта:** рубли (₽)
- **Агенты автономны** — сами принимают решения на основе своего "интеллекта"
- **Обучение** — каждый агент учится на опыте и адаптирует стратегию
- **Экономические институты** — агенты могут создавать "министерства", "банки", "компании"
- **Сделки** — торговля, инвестиции, кредиты, производство

## Стек

- **Backend:** Python 3.11+, FastAPI, SQLAlchemy, asyncio
- **Frontend:** Next.js 14, TailwindCSS, Recharts/D3.js
- **Database:** PostgreSQL 16, Redis (cache + pub/sub)
- **AI:** OpenRouter API (23 модели, free + paid)
- **Deploy:** Docker Compose

## Модели ИИ (роутинг)

| Роль | Модель | Зачем |
|------|--------|-------|
| Простые решения | `qwen3-coder:free` | Быстрые, дешёвые |
| Торговля/рынок | `deepseek-v4-flash` | Аналитика |
| Стратегия | `grok-4.1-fast` | Долгосрочное планирование |
| Исследования | `hunter-alpha:free` | Инновации |
| Reasoning | `sherlock-think:free` | Сложные решения |
| Отчёты | `sonoma-dusk:free` | Текстовые отчёты |
